Iconomus

Placeholder icons in a jiffy (that's fast).

What is this?

Iconomus generates placeholder icons based on attributes you provide. So far, you can specify:

icon color opacity rotation size format

How do I use it?

It's easy! All you have to do is provide an iconom.us URL with the icon attributes you want.

You can use the icons in your HTML or CSS. Check out the examples below, won't you?
