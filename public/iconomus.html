Iconomus

Placeholder icons in a jiffy (that's fast).

What's this for?

Iconomus generates placeholder icons based on attributes you provide. So far, you can specify:

color opacity rotation size format

How do I use it?

It's easy! All you have to do is provide an iconom.us URL with the attributes you want in your HTML or CSS. Like so:

http://iconom.us/tshirt/30x30/F00/

Built with loving tenderness by Ben Johnson and Tack Mobile. The icons are from The Noun Project.
